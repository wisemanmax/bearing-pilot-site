# Bearing — pilot site (preview)

Static preview build of the Bearing pilot site, served via GitHub Pages.

- Every page ships `noindex, nofollow` and is not intended for search indexing.
- This is a **preview** for review, not a public launch under the mark.
- Forms post to a write-only Supabase backend (the publishable key is safe to ship).

Internal strategy, legal, and operations documents are kept in a separate
private project and are intentionally **not** part of this repository.
